Setup

If you've set up for our Intro to R workshop, skip steps 2 and 3.

Mac

  1. Add to Chrome the Selector Gadget plugin
  2. Download and install R
  3. Download and install RStudio
  4. Copy and paste the purple code on the next line into the RStudio console (usually the lower left-hand box), press enter to run
source('https://raw.githubusercontent.com/HoustonUseRs/intro-to-web-scraping/master/scripts/setup.r')
